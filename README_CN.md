# C++ Cmake Template: Tiny

A cmake template for single C++ library.

更多相关内容请移步[AyCmakeTemplate](https://gitee.com/ayin-tech/ay-cmake-template)。

## 适用场景

- 静态库/动态库

- 独立模块

## 快速开始

### 构建

```bash
sh build.sh
sh build.sh Release
```
